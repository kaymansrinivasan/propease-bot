# EST Bot Project

## Overview
The EST Bot is a Python-based application designed to help users manage their time efficiently in Eastern Standard Time (EST). This project facilitates various functionalities related to scheduling, reminders, and time tracking.

## Features
- **Scheduling**: Create and manage schedules with ease.
- **Reminders**: Set up reminders for important tasks and events.
- **Time Tracking**: Track how much time you spend on various activities.

## Installation
To get started with the EST Bot, clone the repository and install the required dependencies:

```bash
git clone https://github.com/kaymansrinivasan/propease-bot.git
cd propease-bot
pip install -r requirements.txt
```

## Usage
After installation, you can run the bot using:

```bash
python est_bot.py
```
